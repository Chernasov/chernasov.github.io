Использование

  • паттерн Page Object – для более гибкого взаимодействия с SUT и сокрытия внутреннего устройства определённой страницы или её виджетов за логическим интерфейсом взаимодействия
  • REST-assured – для тестирования API
  • Selenide – для тестирования GUI
  • Apache Commons – для запроса к БД
  • Docker - для развертывания БД

для тестирования веб-сервиса. Дополнительно реализована возможность запустить в контейнерах две базы данных (Mysql и Postgres) и параметризировать запуск приложения и тестов, передавая в параметрах разные конфигурации для подключения к любой из запущенных баз данных. Проект расположен на GitHub
